# V3 #16 — Security acceptance (#2.5)

Independent security-acceptance pass over the #15 backend, mapping each §3.1/§3.2
design rule and constitution §2 invariant to its evidence. Per the V3 design §10,
this is a dedicated issue so the items can't be silently skipped.

Evidence lives in `taskpaw_v3/tests/test_security.py` (+ the #15 suite). Run:
`uv run pytest taskpaw_v3/tests/test_security.py`.
